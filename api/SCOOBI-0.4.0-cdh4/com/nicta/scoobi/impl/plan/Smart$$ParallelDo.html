ParallelDo

case class ParallelDo[A, B, E](in: DComp[A, Arr], env: DComp[E, Exp], dofn: EnvDoFn[A, B, E], groupBarrier: Boolean, fuseBarrier: Boolean, mA: Manifest[A], wtA: WireFormat[A], mB: Manifest[B], wtB: WireFormat[B], mE: Manifest[E], wtE: WireFormat[E]) extends DComp[B, Arr] with Product with Serializable

The ParallelDo node type specifies the building of a DComp as a result of applying a function to all elements of an existing DComp and concatenating the results.

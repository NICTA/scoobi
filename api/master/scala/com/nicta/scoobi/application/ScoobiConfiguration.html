ScoobiConfiguration

case class ScoobiConfiguration(configuration: Configuration, userJars: Set[String], userDirs: Set[String]) extends Product with Serializable

This class wraps the Hadoop (mutable) configuration with additional configuration information such as the jars which should be added to the classpath.

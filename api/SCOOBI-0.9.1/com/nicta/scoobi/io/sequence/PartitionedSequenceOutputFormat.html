PartitionedSequenceOutputFormat

class PartitionedSequenceOutputFormat[P, K, V] extends PartitionedOutputFormat[P, K, V]

This format creates a new sequence record writer for each different path that's generated by the partition function Each record writer defines a specific OutputCommitter that will define a different work directory for a given key.

All the generated paths will be created under temporary dir/sink id in order to collect them more rapidly with just a rename of directories (see OutputChannel)
